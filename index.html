Aplicación Web React con API de .NET Core

Sportswear eCommerce

Sportswear es un eCommerce de ropa deportiva. El usuario puede recorrer el catálogo para ver los productos extraidos de una API, filtrarlos e ir agregándolos a su carrito para su posterior compra.

Este es un proyecto realizado para una simulación laboral de No Country con un equipo multidisciplinario. La aplicación esta hecha con React y estilizada con Tailwind, aparte hace uso de una API de .NET Core diseñada con Entity Framework por el grupo.
